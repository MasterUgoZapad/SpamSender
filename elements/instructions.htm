Посылка писем

Чтобы послать письмо отдельному сохраненному человеку, перейдите сначала на страницу просмотра адресатов и выберите опцию «послать письмо»

Чтобы отправить письмо группе людей, перейдите на страницу группы и создайте новую. Затем, вы сможете на странице отправки писем выбрать эту группу или на странице просмотра групп выбрать опцию «послать письмо».

Чтобы оправить письмо на почтовый адрес, введите его в поле адреса.

При отправке письма могут быть применены шаблоны. Шаблоны могут содержать специальные конструкции, которые в готовом письме будут заменены соответствующими персональными данными адресатов.

Внимание, при отправки писем на простой почтовый адрес с применением шаблона, данные в шаблоне не будут заполнены.

Адресаты.

На странице просмотра адресатов вы можете увидеть полный список сохраненных контактов.

При клике на контакте вы увидите подробную информацию по нему.

Контакты можно удалять, добавлять, редактировать их данные, посылать им письма.

К каждому контакту соответствует список адресов электронной почты, причем адресов может вообще не быть, так и быть несколько.

Массовый импорт.

Существует возможность импорта данных в *.csv формате.

Первой строкой в файле должны быть имена колонок. Импортируемый файл может содержать следующие колонки:

Название

mail

name

sname

fname

area

town

origin

role

birthdate

Значение

Почта

Имя

Фамилия

Отчество

Округ/область

Город

Происхождение

Статус

Дата рождения

Колонки с другими именами будут проигнорированы.

Так же можно задать значения по умолчанию для всех полей адресата. Значения по умолчанию будут применяться в том случае, когда нужная колонка будет отсутствовать или ее значение будет пустым или не валидным.

Внимание, при загрузке файлов содержащих кириллические символы, убедитесь в верности выбранной кодировки.
